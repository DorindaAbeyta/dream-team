Relations

  • One Profile can write many Posts (1 to many)
  • One Profile can React to many Posts (1 to many)
  • One Profile can Flag many Posts (1 to many)
  • One Hashtag can go to many Posts (1 to many)
  • One Category can have many Posts (1 to many)
  • One Category can have many Hashtags (1 to many)
  • Many Posts can have many Hashtags (many to many)
  • One Post can have many Reactions (1 to many)
  • One Post can have many Flags (1 to many)
  • One Post can have many Categories (1 to many)
  • One Post can have many Hashtags (1 to many)
  • One profile can have many posts. (one-to-many)
  • One post can have many post categories. (one-to-many)
  • One category can have many post-categories. (one-to-many)
  • One profile can have many votes. (one-to-many)
  • One post can receive many votes. (one-to-many)
