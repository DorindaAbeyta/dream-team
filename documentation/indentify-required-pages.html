Static pages: pages that always behave the same no matter who is accessing them or what has been searched for

Dynamic pages: pages that have information that changes based on which entity is being accessed (e.g. a page that shows information for a tweet will have the same architecture, but will have different content depending on which tweet is being viewed)

Examples: Home Page, User Profile, Dashboard,......
